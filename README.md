# Reya Patel - UI & UX Designer Portfolio

A beautiful, responsive single-page portfolio website for UI/UX and graphics designer Reya Patel.

## 📁 Project Structure

```
PORTFOLIO/
├── index.html       # Main HTML file with all sections
├── styles.css       # Complete styling and responsive design
├── script.js        # Interactive features and animations
└── README.md        # This file
```

## 🚀 Getting Started

### Quick Start
1. Simply open `index.html` in your web browser
2. No server or build tools required!

### Windows
- Double-click `index.html` to open in your default browser
- Or right-click → "Open with" → Choose your preferred browser

### Mac/Linux
- Double-click `index.html`
- Or run from terminal: `open index.html` (Mac) or `xdg-open index.html` (Linux)

## 📝 Customization

### Update Contact Information
In `index.html`, modify the contact section:
```html
<p>Email: <a href="mailto:your-email@example.com">your-email@example.com</a></p>
```

### Add Your Projects
Replace placeholder sections with your actual project images and descriptions:
- Update portfolio item titles and descriptions
- Replace placeholder colors with actual project images
- Add project links where appropriate

### Change Colors
In `styles.css`, modify the CSS variables at the top:
```css
:root {
    --primary-color: #8b6f47;
    --dark-brown: #3d2817;
    /* ... other colors ... */
}
```

### Update Navigation Links
Modify the navigation bar in `index.html` to match your sections:
```html
<li><a href="#section-id">Section Name</a></li>
```

## 🚀 Deployment

### Netlify (Recommended - Free)
1. Push files to GitHub
2. Connect to Netlify
3. Deploy automatically

### GitHub Pages (Free)
1. Create a GitHub repository
2. Upload files to `main` branch
3. Enable Pages in repository settings

### Manual Hosting
Upload all three files (index.html, styles.css, script.js) to any web hosting service.
